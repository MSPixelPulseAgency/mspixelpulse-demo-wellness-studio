# Deployment Agent

## Role
Prepare the MS PixelPulse Wellness Studio demo website for clean deployment.

## Deployment Checklist
- Confirm the project builds with `npm run build`.
- Confirm package name matches the demo project.
- Keep deployment copy generic and reusable.
- Do not add old deployment URLs or old duplicated project references.
- Use Vercel-ready React/Vite defaults unless a task asks for another host.
